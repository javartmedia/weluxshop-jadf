# WeluxShop - Beauty Clinic & Skincare E-Commerce

WeluxShop is a modern web application for selling skincare products, booking treatments, managing customers, and running an admin dashboard.

## Features

- 🛍️ Product catalog with categories, brands, and filters
- 📅 Booking system for beauty treatments
- 🛒 Shopping cart & checkout
- 💳 Payment integration (Midtrans/Xendit ready)
- 🎟️ Voucher & promo management
- 👤 Customer accounts with wishlist, orders, points
- 📝 Blog/articles
- 📞 Contact form
- ⭐ Reviews & testimonials
- 🔒 Role-based admin panel with full CRUD
- 📊 Dashboard with sales charts
- 🗄️ Backup & restore
- 🔍 SEO friendly

## Tech Stack

- **Backend:** PHP 8.2+ (MVC), MySQL 8+
- **Frontend:** Tailwind CSS, JavaScript (vanilla)
- **Authentication:** Session-based with password_hash
- **Security:** CSRF, XSS, SQL injection prevention, rate limiting
